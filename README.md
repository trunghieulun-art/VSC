# VSC

## Demo

[![Demo](demo.png)](https://www.youtube.com/watch?v=dQw4w9WgXcQ)

## Setup

### Clone repo

```fish
git clone git@github.com:hnqv313/VSC.git
cd VSC
```

## Run

### Print help

```fish
python main.py --help
```

### Training

```fish
python main.py train --help
```

### Checking

```fish
python main.py check --help
```

### Tuning

Write custom code to evaluate the model. You can see the example in `tune.py`, run `python tune.py`.
